# Brain Later

## Description
A note-taking application that allows users to create, read, update, delete and share notes.

## Features
- User authentication
- Note management
- Responsive design

## Tech Stack
- TypeScript
- React
- Redux Toolkit
- Node.js
- Express
- MongoDB
- JWT
- Bcrypt
- Axios
- Tailwind CSS
- Lucid React
- React Router DOM
